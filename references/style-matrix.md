# Cover Style Matrix

Choose one row before writing prompts. Adapt the object and action to the article; do not combine multiple rows into a collage.

## Article Type Matrix

| Article Type | Cover Promise | Core Object | Xiaoliang Action | Scene Family | Accent |
| --- | --- | --- | --- | --- | --- |
| AI tutorial | "普通人也能照着做" | physical workflow card, notebook, switch, map | placing a card, pressing a button, explaining beside object | teaching desk, clean studio, wall flow | cyan / blue |
| Tool experience | "这个工具到底解决什么" | toolbox, microscope, archive box, spotlight object | opening, uncovering, inspecting, plugging in | studio desk, archive table, lab table | cyan |
| Open-source project | "项目机制被看懂" | metal repo box, branch tags, blueprint, cable hub | connecting cable, opening box, pointing to physical branch markers | maker bench, clean dark table, blueprint desk | blue |
| Product review | "值不值得用" | single product, package, teardown tray | measuring, holding near camera, opening packaging | lightbox, tabletop review, teardown mat | cyan / orange-red |
| Hardware maker | "代码变成实体" | ESP32, LED matrix, sensor, status light | soldering, plugging USB, measuring probe, pressing tiny button | maker desk, parts tray, tool wall | cyan / green |
| Failure review | "问题在哪里" | broken part, warning tag, loose cable, isolated device | circling, pulling cable, exposing broken component | hard side-light desk, teardown tray | orange-red |
| Opinion analysis | "趋势判断" | symbolic object, window light, physical scale, map | standing half-body, resting hand on object, turning to camera | clean studio, wall, window corner | white / blue |
| Reading or knowledge | "资料被看透" | open book, paper stack, archive box, illuminated notes | opening, revealing, leaning in, lifting page | library corner, desk lamp, bookshelf shadow | cyan |
| Creator announcement | "我做了一个东西" | finished build, launch button, labeled box | lifting finished result, pressing launch button, proud reveal | simple studio, maker table | cyan / orange |

## Platform Composition Matrix

| Platform | Layout | Title Zone | Person Zone | Object Zone | Avoid |
| --- | --- | --- | --- | --- | --- |
| WeChat `2.35:1` | left-right tension, diagonal depth, sparse background | left or center-left, 2-3 huge lines | opposite side, readable face, half-body or close medium | foreground or center crossing the frame | tiny title, flat portrait, phone pose, multiple panels |
| Xiaohongshu `3:4` | mobile-first close frame, strong title block, depth object | top 40-50%, 2-4 oversized lines | middle/lower, close enough to recognize | large foreground object or action cue | distant person, small title, screenshot stack, too many captions |

## Pose Rotation

Use at least two changes from the previous successful cover:

| Dimension | Options |
| --- | --- |
| Body posture | seated close-up, standing half-body, side-profile workbench, over-shoulder, low-angle hero, leaning back in realization |
| Hand action | plugging in cable, opening lid/book/box, measuring with probe, pulling loose cable, lifting finished build, pressing one button |
| Camera angle | low foreground, top-down, three-quarter side, close face plus object, wide maker-room, mirror/reflection, doorway reveal |
| Light source | object glow, desk lamp, window light, hard side light, daylight bench, spotlight |

## Title Patterns

Keep the final title short and concrete.

| Pattern | Use When |
| --- | --- |
| `<工具名>\n真能提效吗？` | tool experience or product review |
| `普通人也能用\nAI 做<结果>` | AI tutorial |
| `<工具名>\n把<对象>\n看透了` | analysis/discovery topic |
| `我试了<时间>\n发现一个坑` | failure review |
| `别急着用\n先看这个` | warning or buying advice |
| `<项目名>\n终于能<结果>了` | open-source project or creator announcement |

## Color Guidance

| Accent | Best For | Avoid |
| --- | --- | --- |
| Cyan | discovery, tutorial, data, workflow, "it works" moments | overusing neon cyberpunk |
| Blue | AI, tools, GitHub/open-source, calm analysis | making the whole cover dark blue |
| Orange-red | warnings, failures, strong opinions, problems | cheap sales-ad styling |
| Green/yellow | hardware status, LED results, maker success | random circuit-board collage |
